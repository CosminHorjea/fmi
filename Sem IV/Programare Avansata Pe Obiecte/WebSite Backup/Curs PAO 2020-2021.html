Curs 2020-2021

Bun venit la suportul online pentru cursul de Programare Avansata pe Obiecte (PAO) !

Cine? Cand?

Profesor curs seria 24: Radu Mincu

Orar seria 24: fiecare Joi 12-14

Limbaj - Java

Cursul si laboratorul pentru materia PAO se desfasoara utilizand limbajul Java.

Pentru a scrie programe/aplicatii in Java avem nevoie de doua lucruri:

  1. JDK (Java Development Kit) - contine compilatorul Java.

  2. Un mediu de dezvoltare (IDE) pentru Java. Exemple: NetBeans, Eclipse etc.

Desfasurarea cursului

Componente necesare:

  1. Microsoft Teams

  2. Un browser cu materialul de curs = acest website (selectati cursul din meniul de sus).

  3. Un IDE: Netbeans / Eclipse / etc.

  4. Atentia dvs.

Evaluarea

50% laborator (minim nota 5)

50% examen (minim nota 5)
